Movie Recommender System

This project explores using recommendation systems to predict user ratings for movies and provide movie recommendations. The system uses collaborative filtering based on embeddings of users, ratings, and movies from the MovieLens32M dataset, trained using PyTorch. The model predicts ratings and generates personalized top-10 movie recommendations.
